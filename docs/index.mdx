# Introduction

**UltimateHomeServer** is a production-level home server in a box. Built with [K3s](https://k3s.io/) and [Helm](https://helm.sh/), and fully customizable via the `uhs` CLI.

UHS is a collection of [Helm charts](https://helm.sh/docs/topics/charts/) that are deployed to a Kubernetes cluster. 
The charts are designed to work together to provide a complete home server solution with all of the top self-hosted applications.

## Features

- **Easy to use** - UHS is designed to be easy to use on a single machine. The `uhs` CLI makes it easy to configure and manage your home server.
- **Top self-hosted applications** - UHS includes all of the top self-hosted applications, including Plex, Sonarr, Radarr, and [more](./category/services).
- **Domain with HTTPS** - UHS is designed to be used with a domain name and HTTPS. Access all of your services at `https://service.yourdomain.com`.
- **Amazing Dashboard** - UHS comes packed with [homepage](https://github.com/benphelps/homepage), a highly customizable dashboard for all of your services.
